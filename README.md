# PV Réception Échafaudage

Application web pour la génération de procès-verbaux de réception d’échafaudage.

## Fonctionnalités
- Saisie chantier mobile (QR code)
- Signature tactile
- Vérificateur avec diplôme
- Génération Excel automatisée
- Export PDF
- Interface admin vérificateurs

## Lancer en local

```bash
pip install -r requirements.txt
uvicorn main:app --reload
